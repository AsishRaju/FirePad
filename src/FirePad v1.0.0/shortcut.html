FirePad Keyboard Shorcuts

# Mac Windows/Linux Action Performed
1 ⌘+n ctrl+n create a new FirePad instance.
2 ⌘+s ctrl+s save the current working pad.
3 ⌘+b ctrl+b Bold text via toggle or selection.
4 ⌘+i ctrl+i Italic text via toggle or selection.
5 ⌘+u ctrl+u Underline text via toggle or selection.
6 ⌘+h ctrl+h Highlight text by selection.
7 ⌘+` ctrl+` Strike text via toggle or selection.
8 ⌘+f ctrl+f Find text.
9 ⌘+up ctrl+up Superscript text via toggle or selection.
10 ⌘+down ctrl+down Suberscript text via toggle or selection.
11 ⌘+shift+c ctrl+shift+c Allign text at center.
12 ⌘+shift+x ctrl+shift+x Allign text at left.
13 ⌘+shift+v ctrl+shift+v Allign text at right.
14 ⌘+tab ctrl+tab Insert tab spaces.
15 ⌘+1 ctrl+1 Insert H1 styling.
16 ⌘+2 ctrl+2 Insert H2 styling.
17 ⌘+3 ctrl+3 Insert H3 styling.
18 ⌘+4 ctrl+4 Insert H4 styling.
19 ⌘+5 ctrl+5 Insert H5 styling.
20 ⌘+6 ctrl+6 Insert H6 styling.
21 alt+c Insert code styling.
22 shift+tab Remove tab spaces.
23 alt+1 Insert ordered list.
24 alt+. Insert unordered list.
25 alt+l Insert line.
26 alt+d Insert date.
27 alt+a Insert link.
28 alt+t Insert todo not-done item.
29 alt+y Insert todo done item.
30 alt+2 Enable smaller font size.
31 alt+3 Enable small font size.
32 alt+4 Enable large font size.
33 alt+5 Enable larger font size.
34 alt+0 Enable Default styling/Reset styling/Reset Font size.
