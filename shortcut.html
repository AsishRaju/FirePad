FirePad Keyboard Shorcuts -Windows

# Key Combination Action Performed
1 ctrl+n create a new FirePad instance.
2 ctrl+s save the current working pad.
3 ctrl+b Bold text via toggle or selection.
4 ctrl+i Italic text via toggle or selection.
5 ctrl+u Underline text via toggle or selection.
6 ctrl+h Highlight text by selection.
7 ctrl+` Strike text via toggle or selection.
8 ctrl+f Find text.
9 ctrl+up Superscript text via toggle or selection.
10 ctrl+down Suberscript text via toggle or selection.
11 ctrl+shift+c Allign text at center.
12 ctrl+shift+x Allign text at left.
13 ctrl+shift+v Allign text at right.
14 ctrl+tab Insert tab spaces.
15 ctrl+1 Insert H1 styling.
16 ctrl+2 Insert H2 styling.
17 ctrl+3 Insert H3 styling.
18 ctrl+4 Insert H4 styling.
19 ctrl+5 Insert H5 styling.
20 ctrl+6 Insert H6 styling.
21 alt+c Insert code styling.
22 alt+tab Remove tab spaces.
23 alt+1 Insert ordered list.
24 alt+. Insert unordered list.
25 alt+l Insert line.
26 alt+d Insert date.
27 alt+a Insert link.
28 alt+t Insert todo not-done item.
29 alt+y Insert todo done item.
30 alt+2 Enable smaller font size.
31 alt+3 Enable small font size.
32 alt+4 Enable large font size.
33 alt+4 Enable larger font size.
34 alt+0 Enable Default styling/Reset styling/Reset Font size.
